# Commodities-Live-Scraper
Overview: a.This Python project fetches live commodity prices using yfinance and records them to a CSV for historical analysis. | b.Tracked commodities: Gold, Silver, Crude Oil, Natural Gas, Copper.

Features: a.Live data scraping from yfinance | b.Timestamped output for tracking | c.Historical CSV storage for long-term trend analysis | d.Automated fetching at configurable intervals (default: 12 hours)

Requirements: a.Python 3.8+ | b.yfinance library

Install dependencies: pip install yfinance

Notes: a.Graphs and trend analysis become meaningful as data accumulates over time. | b.Fetch interval can be adjusted for finer or coarser data resolution.
